This is a rough rendering of a page from the old Prevayler wiki. Please see the new wiki for current documentation.

Prevayler's latest development code is pretty stable because we use a lot of automated tests (JUnit).

It can be accessed through CVS:
http://sourceforge.net/cvs/?group_id=36113

CVS is simple but touchy. If you are new to CVS, try having a friend help you with it. That will considerably reduce your learning time.
