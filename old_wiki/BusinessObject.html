BusinessObject

This is a rough rendering of a page from the old Prevayler wiki. Please see the new wiki for current documentation.

An object that lives in BusinessObjectParadise and is worried only about performing calculations and storing information. It is Deterministic and does not access any external device (keyboard, clock, network, etc) directly or indirectly, therefore.

It represents a solution for a problem of EssentialDifficulty.
Contrast with: CommunicationObject.
