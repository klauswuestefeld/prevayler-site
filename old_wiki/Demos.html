I've wrapped up a simple demo on how to query objects using XPath (with the JXPath package by Jakarta).

You'll need Prevayler and the commons-jxpath.jar in your CLASSPATH for this demo to work.

Binaries are here: http://www.prevayler.org/upload/cv/jxpath-bin.jar
And here's the source: http://www.prevayler.org/upload/cv/jxpath-src.jar

PS: JacobKjome had a lot of problems running this demo regarding to date formats. If you're under a en_US locale, try  using "MM/DD/YY HH:MM (AM|PM)". You can read more about it at http://www.javaworld.com/javaworld/jw-02-1999/jw-02-internationalize-p2.html.
