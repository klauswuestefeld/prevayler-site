DoesPrevalenceOnlyWorkInJava

This is a rough rendering of a page from the old Prevayler wiki. Please see the new wiki for current documentation.

No.

You can use any language for which you are able to find or build a serialization mechanism. In languages where you can directly access the system's memory and if the business objects are held in a specific memory segment, you can also write that segment out to the snapshot file instead of using serialization.

See: PrevaylerPortsToOtherLanguages.
