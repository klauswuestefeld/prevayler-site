LogYourCommands

This is a rough rendering of a page from the old Prevayler wiki. Please see the new wiki for current documentation.

If you're going to write a heavy prevalent system, or are trying to debug one, an interesting thing to implement is a good logging system. This can be accomplished very easily with Prevayler if you implement a descriptive toString() for your commands and log them directly into a text file, with a timestamp. An example:

[2002-08-25 14:38:24] AddList: Adding list 'Prevayler Team'
[2002-08-25 14:38:24] AddUser: Adding user 'Carlos' to 'Prevayler Team' list
[2002-08-25 14:38:26] AddUser: Adding user 'Klaus' to 'Prevayler Team' list
[2002-08-26 17:21:43] AddList: Adding list 'Prevayler Users'
[2002-08-26 17:25:12] AddUser: Adding user 'John Doe' to 'Prevayler Users' list

If command-logging and snapshot callbacks are implemented in the main Prevayler tree (see HistoryLogging), you'll also be able to track on what command-logs are your commands. Then, it will be possible to create a log file like this:

[2002-08-25 14:38:24] 000000000015: AddList: Adding list 'Prevayler Team'
[2002-08-25 14:38:24] 000000000016: AddUser: Adding user 'Carlos' to 'Prevayler Team' list
[2002-08-25 14:38:26] 000000000017: AddUser: Adding user 'Klaus' to 'Prevayler Team' list
[2002-08-26 17:21:43] 000000000018: AddList: Adding list 'Prevayler Users'
[2002-08-26 17:25:12] 000000000018: AddUser: Adding user 'John Doe' to 'Prevayler Users' list

-- CarlosVillela
