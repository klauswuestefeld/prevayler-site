InmarsatCallAnalysisProject

This is a rough rendering of a page from the old Prevayler wiki. Please see the new wiki for current documentation.

This Internet deployed application allows Inmarsat resellers to perform detailed analysis of D91 and CDR formatted calls attributed to them, allowing a unique insight into client usage patterns.

This system is currently being used by over 125+ Inmarsat clients globally

Key technologies used are:
-Prevayler Java persistent database
-Java
-Velocity HTML/Java templating
-Resin Servlet container
This application was developed and deployed into production in only 8 weeks, and is currently hosted and supported by us as part of our Bureau services.
