ObjectServer

This is a rough rendering of a page from the old Prevayler wiki. Please see the new wiki for current documentation.

The heart of a truly object-oriented system.

It is a cohesive server with the same function as both the DataServer and ApplicationServer (from a traditional three-tier architecture) combined into a single process, which boosts performance and greatly simplifies the architecture and administration of any system.

See: ScalabilityTestResults.
