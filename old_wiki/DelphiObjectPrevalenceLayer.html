DelphiObjectPrevalenceLayer

This is a rough rendering of a page from the old Prevayler wiki. Please see the new wiki for current documentation.

The Delphi Object Prevalence Layer is a framework to alow all objects to reside in memory and persists to binary snapshots for backups. This eliminates the need of databases or the use of Object Persistence Layers (OPL).

http://sourceforge.net/projects/dopl
